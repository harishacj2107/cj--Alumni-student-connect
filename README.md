# Alumni Student Connect

## Overview

Alumni Student Connect is a Flask-based web application designed to bridge the gap between alumni and students. The platform enables secure user registration, profile management, real-time messaging, and role-based access for administrators, alumni, and students.

## Features

- Secure user registration and login.
- Role-based access for Admin, Alumni, and Students.
- Profile creation and management.
- Real-time messaging between alumni and students.
- Interactive dashboards for different user roles.
- Password security and user authentication.
- CRUD operations for managing user information.

## Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- SQLite
- Jinja2

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/harishacj2107/harisha-Alumni-student-connect.git
```

### 2. Navigate to the project directory

```bash
cd harisha-Alumni-student-connect
```

### 3. Install the required packages

```bash
pip install -r requirements.txt
```

### 4. Run the application

```bash
python app.py
```

### 5. Open your browser

```
http://127.0.0.1:5000
```
